**CashTrack** is an intelligent financial management platform designed specifically for Rutgers students. 
It provides real-time spending analytics, personalized financial coaching, and actionable insights to help students make smarter financial decisions during their college journey.

**üéØ Problem Statement -**

College students often struggle with:

- Overspending without realizing it until it's too late
- Lack of visibility into spending patterns across categories
- No personalized guidance on budget optimization
- Difficulty tracking multiple payment methods and transactions

CashTrack solves these problems by providing a unified dashboard with AI-powered coaching that helps students understand, track, and optimize their spending habits.

**‚ú® Key Features**
1. Real-Time Spending Dashboard

- Live telemetry of all campus wallet transactions
- Interactive filters by date, category, and amount
- Beautiful, modern UI with glassmorphic design

2. Comprehensive Analytics

- Category-wise spending breakdown with visual charts
- Daily and weekly spending trends
- Weekday pattern analysis to identify high-spending days
- Key performance indicators (KPIs): total spent, transaction count, averages

3. RU Money Coach (AI-Powered Advisor)

- Personalized financial recommendations based on:

- Residence type (Dorm, Commuter, Off-Campus)
    Major and class year
    Spending patterns and habits

- Chat-style interface for friendly, actionable advice
- Context-aware tips (e.g., meal plan optimization, book-buying strategies)

4. Privacy-First Transaction Logs

- Detailed transaction history (visible only for individual students)
- Filter and search capabilities
- Export-ready format for monthly reviews

5. Student Profile Integration

- Links spending data with student demographics
- Personalized insights based on major, residence, and interests
